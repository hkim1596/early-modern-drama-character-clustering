Character Clustering in Early Modern English Drama

An interactive map of dramatic-character archetypes built from the full speech of every named character in the early modern English repertoire. Each point is a character; color encodes cluster.

Cluster maps

Baseline preset

UMAP n_neighbors=15, HDBSCAN min_cluster_size=20. The recommended view.

Less-outliers preset

UMAP n_neighbors=35, HDBSCAN min_cluster_size=30, min_samples=10. Smoother neighborhood graph.

How to read the map

Each point is a single character. Colors encode cluster (HDBSCAN). Hover for play, author, date, genre, role, cluster, and the cluster's top distinguishing words. Use the dropdown menus at the right to highlight a single decade, genre, author, play, play type, or theater at a time.

About

Characters were extracted from EEBO-TCP; play metadata is merged from Wiggins & Richardson's British Drama: A Catalogue of Plays and from DEEP. Each character's full speech is encoded with the Alibaba-NLP/gte-Qwen2-1.5B-instruct long-context sentence model — no chunking. Clustering uses UMAP (5-D, cosine) then HDBSCAN; the visible 2-D map is a separate UMAP projection of the same embeddings.
